Source `skills/rabbit-spec-create/SKILL.md` (v1.3.0 -> v1.4.0) Step 1/Step 4 updated to document and surface the stderr note; `scripts/dispatch-spec-create.py` docstring (v1.1.1 -> v1.2.0) updated.
